# VRBO via Hostaway

## VRBO vs. Expedia: Key Differences

| | Expedia | VRBO |
|---|---|---|
| Focus | Hotels + bundles (vacations, flights) | Vacation rentals exclusively |
| Guest type | Business travelers, package deal seekers | Families, groups, vacation rental guests |
| Hostaway connection | Separate integration | Separate integration |
| Content sync | No (manage in EPC) | No (manage in VRBO dashboard) |
| Booking type | Instant book | Instant book or request-to-book |
| Commission | 15–18% per booking | 8% per booking OR $499–$699/yr + 3% |

**They are owned by the same parent company (Expedia Group) but are completely separate channels in Hostaway.** Connecting one does not connect the other.

If Expedia's onboarding redirects you to VRBO, that is intentional — Expedia routes vacation rental hosts to VRBO for the dedicated STR audience.

---

## VRBO Commission Structure

**Pay-per-booking (recommended for most hosts):**
- 5% booking fee
- 3% payment processing fee
- **Total: 8% per booking**
- If using a PMS (like Hostaway): additional 5% fee may apply on VRBO-affiliated sites, bringing total closer to 8–13%

**Annual subscription (high-volume hosts):**
- $499–$699/year (varies)
- 3% payment processing fee per booking
- Break-even ~$10,000–$14,000 in annual VRBO revenue

## VRBO Connection in Hostaway

Similar process to Expedia:
1. Create listing in VRBO owner dashboard first
2. Connect via Hostaway Channel Manager → VRBO
3. Map listings by VRBO property ID
4. All content updates happen in VRBO dashboard, not Hostaway

---

## VRBO Booking Types

- **Instant Book**: Guest books immediately, host gets notification
- **Request-to-Book**: Host has 24 hours to accept/decline; declining hurts your ranking
- Setting is configured per listing in VRBO dashboard
- Instant book is strongly recommended for visibility

---

## VRBO Guest Communication in Hostaway

- VRBO messaging syncs to Hostaway unified inbox
- Automated Hostaway messages work with VRBO bookings
- See Hostaway support article: "Common Messaging Issues with Vrbo on Hostaway"

---

## VRBO-Specific Considerations

- VRBO guests typically book further in advance and for longer stays than Airbnb
- Guest reviews are highly weighted in VRBO's search algorithm
- VRBO charges guests a separate service fee on top of your nightly rate
- Annual subscribers get the VRBO Premier Host badge sooner (better rankings)
- VRBO does not allow communication off-platform until after booking confirmation

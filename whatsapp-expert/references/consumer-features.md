## Messaging

- End-to-end encrypted text, voice messages, images, video, documents, contacts, location
- File sharing up to **2 GB per file**
- Full-resolution photo and video sharing
- **Formatting**: bold (`*text*`), italic (`_text_`), strikethrough (`~text~`), monospace (` ```text``` `), bulleted/numbered lists
- **Reactions**: tap-and-hold to react with any emoji
- **Edit messages**: up to 15 minutes after sending
- **Pin messages**: up to 3 pinned per chat (tap-and-hold → pin)
- **Reply to specific messages**: inline quote thread
- **Star messages**: bookmark for later (Starred Messages section)
- **Search within conversations**: tap search icon in chat
- **Forward messages**: "Forwarded" label applied; mass-forwarded messages limited to 1 recipient at a time

## Calls

- HD voice and video calls — 1-on-1 and group
- **Group calls**: up to 32 participants
- Video call features: filters, virtual backgrounds, screen sharing
- **Call links**: shareable URLs to join a call without requiring contacts to be saved
- **Voice Chat** (in groups): open ambient audio channel (walkie-talkie style for large groups; members join/leave freely)
- Noise cancellation on supported devices

---

## Groups

- Up to **1,024 members** per group
- **Admin controls**: restrict sending to admins only; control who can edit group info
- **Group polls**: create multi-option polls; single or multiple choice
- Group descriptions, icons, invite links with optional expiry
- **Disappearing messages**: toggle at group level with timer options
- **Voice Chat**: open audio channel within the group (ambient conversation)
- **New member history**: admins choose what message history new joiners can see
- Group calling: up to 32 participants
- Admin-only mode: only admins can send messages (good for announcements)

---

## Communities

- Launched 2022; umbrella structure grouping multiple related groups under one entity
- Designed for: schools, workplaces, local organizations, neighborhoods
- **Announcement channel**: community admins broadcast to all member groups simultaneously (admin-only)
- 300 million+ messages per day across communities (2025)
- 120 million+ active communities globally
- Members can see all groups in a community but join selectively
- Community admins manage group membership from a central dashboard
- Distinct from Channels: Communities are collaborative; Channels are broadcast-only

---

## Channels

- One-way broadcast from admin to followers (followers cannot reply publicly)
- No follower limit
- Follower identities are **private** from the channel admin
- Accessible via the **Updates tab** in WhatsApp (separate from Chats)
- Admins can send: text, photos, videos, stickers, polls
- Content auto-deletes after **30 days** by default
- **Public channels**: discoverable in the WhatsApp channel directory
- **Private channels**: invite-only
- Followers can react to channel posts (emoji reactions only)

## Status / Stories

- Photo, video, GIF, or text updates visible to contacts for **24 hours**
- **Privacy controls**: all contacts / select contacts / all except... (custom exclusion lists)
- Viewers can reply to your status via direct message
- **View Once status**: disappears after viewer opens it
- **AI-generated images**: create images for status using Meta AI (generative image models)
- Animate photos into short video clips using Meta AI

## Privacy & Security

### Visibility Controls
| Setting | Options |
|---------|---------|
| Last seen | Nobody / Contacts / Contacts except... / Everybody |
| Profile photo | Nobody / Contacts / Contacts except... / Everybody |
| About (bio) | Nobody / Contacts / Contacts except... / Everybody |
| Online status | Nobody / Same as Last Seen |
| Read receipts (blue ticks) | On / Off (disabling also hides others' receipts) |
| Groups: who can add you | Everybody / Contacts / Contacts except... / Nobody |

### Security Features
- **Two-step verification**: PIN-based additional security layer for account registration
- **Passkey support** (FIDO2): biometric/device-based login without SMS OTP (2023+)
- **Chat Lock**: locks specific chats behind a separate PIN; hides them from the main chat list
- **Secret Code**: set a unique code to reveal Chat Lock folder; hidden from app if code isn't entered
- **Advanced Chat Privacy**: prevents export of chat content and Meta AI processing of messages
- **Silence unknown callers**: automatically silence calls from numbers not in contacts
- **IP address protection in calls**: routes calls through WhatsApp servers to hide real IP address
- **Blocked contacts list**: block specific numbers from contacting you

### Account Security
- End-to-end encryption for all messages and calls by default
- Security codes (Safety Numbers): verify encryption with contacts
- Two-device confirmation for linking companion devices

## Disappearing Messages & View Once

### Disappearing Messages
- Available in 1-on-1 chats and groups
- **Timer options**: 24 hours, 7 days, 90 days (plus shorter: 1 hour, 12 hours)
- **Default timer**: set a global default for all new chats (Settings → Privacy → Default Message Timer)
- Messages deleted when timer expires whether read or unread
- Does not prevent screenshots

### View Once
- Send photos, videos, or voice messages that disappear after opening once
- Screenshots blocked on Android while viewing View Once media
- Auto-deleted if not opened within **14 days**
- Voice message View Once: auto-deletes after a single play
- Separate from Disappearing Messages (View Once is per-message, not timer-based)

## Web & Desktop

- **WhatsApp Web**: browser-based at web.whatsapp.com — no app install required
- **Desktop apps**: native apps for Windows and macOS
- **Multi-device**: up to 4 linked companion devices; phone does NOT need to be online
- All core features available: messaging, voice/video calls, Chat Lock, most privacy settings
- Messages sync across all linked devices
- Link devices: Settings → Linked Devices → Link a Device (scan QR code)
- Desktop beta program available for testing unreleased features

---

## Meta AI Integration

- Built-in AI assistant accessible via the search bar and directly in chats
- **@Meta AI** in groups: mention to ask questions within a group conversation
- **Image generation**: create images directly in chat and for Status updates
- **Group chat summaries**: summarize unread messages in busy group chats
- **Draft messages**: AI-assisted message drafting
- Data note: Meta AI interactions are not end-to-end encrypted

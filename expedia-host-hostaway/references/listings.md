# Expedia Listing Content Requirements

## What to Manage in Expedia Partner Central (NOT Hostaway)

All listing content must be created and updated directly in EPC:
- Property photos
- Description / property name
- Amenities (room-level and property-level)
- Maximum guest capacity
- House rules and policies
- Property type and category

Hostaway does not push content to Expedia. Content updates made in Hostaway are not reflected on Expedia.

---

## Content Score

Expedia's **Content Score** (0–100) directly impacts search ranking and visibility. Higher score = better placement. Expedia evaluates:

1. **Property Amenities** — pool, parking, recreation facilities, local attractions
2. **Room Amenities** — TVs, views, bathroom features, climate control, accessibility features
3. **Policies & Deposits** — cancellation policy, deposit requirements
4. **Photos** — quantity, quality, coverage

Access your Content Score in EPC under the property performance dashboard.

## Photo Requirements

| Requirement | Detail |
|---|---|
| Minimum | 10 photos |
| Recommended | 20–30 depending on property size |
| Resolution | Minimum 2,880 pixels wide |
| Orientation | Horizontal preferred |
| Per unit type | 4 photos (bathroom, kitchen, view, bedroom) |
| Exterior | 1 shot showing building and surroundings |
| Entrance | 1 entrance/reception photo |
| Amenities | 1 photo per major amenity (pool, gym, dining area) |

**Impact:** Properties with unique photos per room type average up to 11% higher conversion rates.

Style tips: well-lit, tidy, uncluttered. Professional photography strongly recommended.

---

## Amenities Strategy

Travelers filter search results by amenities. If an amenity is not listed, the property won't appear in filtered results.

Two categories:
- **Room amenities**: in-unit features (TV, kitchen appliances, A/C, bathtub, accessibility)
- **Property amenities**: shared or building-level (pool, gym, BBQ, parking, shuttle)

List every amenity the property actually has — omissions hurt visibility, not accuracy.

---

## Description Best Practices

- Lead with the property's strongest differentiators
- Include neighborhood/location context
- Mention nearby attractions and transport
- Accurately describe sleeping arrangements
- Keep tone professional (Expedia's audience expects hotel-adjacent content)

---

## Property Clustering

If you have multiple units at the same address, Expedia supports **clustering** them under one listing. This increases visibility and traveler confidence, improving conversion rates. Set this up directly in EPC.

---

## Visibility Factors (Search Ranking)

Expedia's algorithm uses two main signals:

1. **Offer Strength**: listing completeness, competitive pricing, calendar availability
2. **Guest Experience**: review scores, refund/relocation rates, complaint volume

Poor guest experience scores → reduced visibility → potential delisting.

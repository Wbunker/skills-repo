# Listings & Inventory Management

## My Properties Page

The **My Properties** page is the main inventory hub. Each listing shows:
- Listing ID / External ID
- Plan type (Pay-Per-Booking, Professional, Unassigned, Expired)
- Status

### Listing Statuses

| Status | Meaning |
|--------|---------|
| **Active** | Live and bookable on Expedia/Vrbo |
| **Incomplete** | Missing required content — not published |
| **Inactive** | Manually deactivated by host |
| **Restricted** | Platform-restricted (policy violation or review) |
| **Pending Publication** | Submitted, awaiting platform review |
| **Archived** | Removed from active management; not visible to guests |

### Filtering & Organizing

- Filter by status, tags, or plan type
- **Tags:** Custom groupings (e.g., "riverfront," "cabin row," "pet-friendly") — add/remove via bulk actions
- **Search bar:** Find listings by name or ID

---

## Listing Minimum Requirements

All of the following must be present before a listing can become Active:

- [ ] At least **6 photos**
- [ ] Headline: **20+ characters**
- [ ] Description: **400+ characters**
- [ ] At least **1 bedroom** and **1 bathroom** defined
- [ ] Valid physical address
- [ ] Accurate rates and calendar
- [ ] A cancellation policy selected

Any single missing item keeps the listing in **Incomplete** status.

---

## Bulk Management Actions

Select multiple listings on My Properties to access:

- **Change Tags** — add or remove organizational tags
- **Archive / Unarchive** — remove from or restore to active management
- **Convert to Pay-Per-Booking** — change plan type
- **Download CSV** — export full inventory list for offline review

## Content Score

**Location:** Property Details section of the listing

Scored 0–100. Sub-scores:

| Sub-score | What It Measures |
|-----------|-----------------|
| Property Amenities | Campground-level amenities (pool, rec facilities, parking, etc.) |
| Room Amenities | In-cabin features (TV, kitchen, climate control, etc.) |
| Policies & Deposits | Cancellation policy, security deposit, house rules |
| Photos | Quantity, quality, and coverage |

A high overall score with a weak sub-score still impacts search ranking for that sub-category.

---

## Offer Strength Score

**Location:** Visibility Performance page

Composite metric covering:
- Content completeness
- Rate competitiveness
- Calendar availability openness

Improving any of the three components raises the score and search position.

---

## Clustering Multiple Units

If multiple cabin units share the same address, they can be clustered under a single listing to consolidate availability, bookings, and reviews — increasing search visibility versus having separate low-review listings.

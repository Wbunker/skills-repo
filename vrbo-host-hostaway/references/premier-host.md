# VRBO Premier Host Program

## What It Is

Premier Host is VRBO's performance recognition program. Properties that qualify earn a **Premier Host badge** visible on search results and listing pages, boosting credibility and bookings.

The program is **free** — no cost to apply or maintain once you qualify.

---

## Current Requirements (2025)

| Metric | Threshold |
|---|---|
| Average review score | 4.4+ (stars) |
| Booking acceptance rate | 95%+ |
| Host-initiated cancellation rate | ≤1% |
| Minimum bookings | 5 bookings OR 60 booked nights in last 12 months |
| Response time | Within 1 hour (business hours); never exceed 24 hours |
| Identity verification | Required |

Performance is assessed quarterly by VRBO.

---

## 2026 Changes (Important)

Starting early 2026, VRBO is transitioning from **host-level** to **listing-level** Premier Host recognition:

- Premier Host status will be evaluated and assigned **per listing**, not per account
- Standards are becoming stricter (acceptance rate 99%+, zero host cancellations, review score 4.6+)
- Hosts who achieved Premier status in 2025 do **not** automatically retain it in 2026
- Each listing must independently meet the new thresholds

Plan ahead: ensure each listing meets the higher standards well before 2026 review periods.

---

## Benefits

**Visibility:**
- Premier Host badge on all listings in your account (current; listing-level in 2026)
- Eligible for VRBO's **Boost program** — improves search position for select dates across your account
- Listings appear when travelers use the **Premier Host filter** in search

**Performance data (Premier vs. non-Premier):**
- Average daily rate (ADR): 7.6% higher ($395 vs. $367)
- Revenue per available room (RevPAR): 8% higher
- Average review score: 4.82 vs. 4.68

**Support:**
- Priority support with 24/7 access to dedicated VRBO team

**Marketing:**
- Eligible for VRBO email campaigns, social media features, and press releases reaching millions of travelers

## How to Qualify via Hostaway

Hostaway helps with several Premier Host metrics:

- **Response time**: Use Hostaway automated messages to send immediate booking confirmations and inquiry responses
- **Cancellation rate**: Hostaway's channel management reduces double-bookings that cause host cancellations
- **Review score**: Hostaway automated post-stay messages can prompt guests to leave reviews
- **Acceptance rate**: Enable Instant Book to remove manual accept/decline from the equation

## Maintaining Status

- Review VRBO performance dashboard quarterly
- Keep calendar updated (at minimum every 60 days — see `references/optimization.md`)
- Respond to all inquiries within 1 hour
- Never cancel on guests unless absolutely unavoidable
- Deliver stays that match the listing description exactly

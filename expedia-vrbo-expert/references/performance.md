# Performance & Analytics

## Analytics Dashboard

**Location:** Performance tab

### Core Metrics

| Metric | What It Measures |
|--------|-----------------|
| Visibility | Search impressions — how often the listing appeared in search results |
| Conversion | Ratio of bookings to listing page views |
| Guest Value | Average room revenue vs. prior year and competitive set |
| Occupancy Rate | Percentage of available nights booked |
| ADR | Average Daily Rate |

All metrics include year-over-year comparisons and can be benchmarked against a defined competitive set.

---

## Competitive Set

Define **5–19 competitor properties** for benchmarking. The comp set is used by:
- The analytics dashboard (Guest Value comparison)
- **Rev+** (competitor rate analysis)

Set up or update the competitive set within the Performance or Rev+ section.

---

## Rev+ / MarketMaker (Free Demand Tool)

**Rev+** (Expedia Partner Central) and **MarketMaker** (Vrbo dashboard) are the same underlying tool with different names per portal. Both are free.

### Features

| Feature | Description |
|---------|-------------|
| Price Calendar | Your rates vs. competitors for the next 90 days |
| Market Demand Score | Demand forecast based on Expedia Group booking data |
| Daily Market Alerts | Notifications when market pricing shifts |
| Competitive Price Grid | Competitor pricing trends over time |
| Occupancy & Compression | Real-time local occupancy signals |

**Use case:** Review the Price Calendar weekly to identify dates where the property is priced above or below market, and adjust rates or activate Accelerator accordingly.

## Premier Host Program (Vrbo)

Vrbo's recognition program for top-performing hosts. Status unlocks search visibility benefits and trust badges.

**Location:** Performance page > Premier Host Scorecard (downloadable report available)

### Minimum Requirements to Qualify

| Requirement | Threshold |
|-------------|-----------|
| Review score | 4.6 or higher |
| Booking acceptance rate | High |
| Cancellation rate | Low |
| Response rate | High |

The Scorecard tracks progress toward each requirement. Check it regularly to identify which metric needs improvement.

## Performance Milestones

A tiered achievement system within the Vrbo dashboard (widget on main dashboard).

- Completing milestones unlocks benefits including:
  - **"Top 1%"** badge displayed on the listing
  - Increased search visibility
- Progress is tracked per listing

Vrbo began rolling out a **personalized coaching experience** in early 2026. It provides:
- Actionable recommendations tailored to each property's performance gaps
- Insights on specific improvements (e.g., "add bathroom photos," "enable Members Only rate")

Check the Performance tab or dashboard notifications for coaching recommendations.

## Search Ranking Factors

Two composite scores drive search position:

### Offer Strength Score
- Content completeness (photos, amenities, description)
- Rate competitiveness vs. market
- Calendar availability (how many dates are open)

### Guest Experience Score
- Relocation/refund rates (lower = better)
- Guest review ratings
- Response rate and speed

Both scores are visible on the Visibility Performance page and in the Premier Host Scorecard.

## Facebook Pages

Pages are public profiles for businesses, organizations, brands, and public figures.

### Page Types
- **Business Page**: For companies, brands, and organizations
- **Creator Page**: For public figures, artists, influencers

### Creating a Page
1. facebook.com/pages/create
2. Choose type → Add name, category, description
3. Upload profile photo (170×170px) and cover photo (820×312px)
4. Add contact info, website, hours, location

### Professional Dashboard
Accessible from your Page — your command center:
- **Page Performance**: Reach, engagement, follower growth
- **Content**: Post performance, top posts, scheduling queue
- **Audience**: Demographics, follower map, new followers
- **Monetization**: Stars earned, in-stream ads revenue, subscriptions

### Page Features
| Feature | Description |
|---|---|
| **Posts** | Standard feed posts; can schedule in advance |
| **Stories** | Ephemeral 24-hour content for followers |
| **Reels** | Short-form video, highest organic reach |
| **Live** | Real-time broadcast to followers |
| **Events** | Create and promote events with ticketing |
| **Services** | List services (for service businesses) |
| **Shop** | List products; integrate with Shopify/WooCommerce |
| **Offers** | Create coupons/promotions for followers |
| **Reviews** | Customer reviews (can disable) |
| **Appointments** | Built-in booking via Facebook |

### Page Roles
- **Admin**: Full control (settings, roles, ads, content)
- **Editor**: Create/publish content, respond to messages
- **Moderator**: Respond to comments/messages, send ads
- **Advertiser**: Create and manage ads only
- **Analyst**: View insights only

### Growing a Page
- Post consistently: 3–5x/week minimum
- Respond to comments within 1 hour (boosts reach)
- Use native video/Reels for maximum reach
- Run "Page Like" campaigns via Ads Manager for follower growth
- Cross-promote on personal profile and other Pages
- Invite friends who've reacted to posts to like the Page

### Page Messaging
- Response time badge: Reply within 15 min → "Very Responsive" badge
- Set up automated responses for common questions (Settings → Messaging)
- Use Inbox to manage Page messages, post comments, and Instagram DMs in one place

## Facebook Groups

Groups are community spaces — can be public or private.

### Group Types
- **Public**: Anyone can see the group, members, and posts
- **Private**: Only members can see posts; joining requires approval or invite

### Creating a Group
1. Groups → + Create Group
2. Name, privacy setting (public/private)
3. Add members → Customize cover photo, description, rules

### Group Features
| Feature | Description |
|---|---|
| **Posts** | Text, photo, video, poll, file uploads |
| **Events** | Group-specific events |
| **Group Reels** | Short-form video posts within the group |
| **Community Chat** | Real-time chat rooms within the group |
| **Rooms** | Audio/video rooms for group members |
| **Files** | Upload and share documents |
| **Polls** | Create polls for member voting |
| **Buy and Sell** | Enable marketplace listings within the group |
| **Nicknames** | Members can set display names within the group (2026) |

### Group Admin Tools
- **Member Requests**: Approve/deny with screening questions (up to 3 questions)
- **Group Rules**: Up to 10 rules members must acknowledge
- **Moderation Alerts**: Auto-flag content with certain keywords
- **Admin Assist**: Auto-approve/decline based on preset criteria
- **Membership Pausing**: Temporarily limit new posts from low-quality accounts
- **Insights**: Post reach, engagement rates, top contributors

### Group Monetization
- **Paid Groups**: Charge membership fees (via Subscriptions)
- **Creator Bonus**: Meta pays creators for cross-posting Reels to Groups

### Effective Group Management
- Set clear rules and enforce consistently
- Use pinned posts for announcements
- Schedule posts for consistent engagement (3–5x/week)
- Feature members and welcome new members weekly
- Use the Admin Assist to auto-remove posts with prohibited keywords

## Meta Business Suite

Unified dashboard for managing Facebook and Instagram presence.

**Access:** business.facebook.com or Meta Business Suite app

### What It Manages
- Multiple Facebook Pages and Instagram accounts from one place
- Content scheduling across both platforms simultaneously
- Unified inbox (Facebook + Instagram messages and comments)
- Ads (creates and monitors campaigns — links to Ads Manager for advanced setup)
- Insights across both platforms

### Key Sections
| Section | Purpose |
|---|---|
| **Home** | Overview dashboard with recent activity and alerts |
| **Inbox** | Unified messages + comments (Facebook + Instagram) |
| **Posts & Stories** | Create, schedule, and manage content |
| **Reels** | Create and schedule Reels for both platforms |
| **Ads** | Quick-create ads; links to full Ads Manager |
| **Insights** | Performance metrics: reach, engagement, audience |
| **Commerce Manager** | Manage Facebook/Instagram Shop catalog and orders |
| **Events Manager** | Set up Facebook Pixel and Conversions API |
| **Business Settings** | Add people, assets, ad accounts, and Pages |

### Business Settings Hierarchy
```
Business Portfolio (top level)
├── Ad Accounts
├── Pages
├── Instagram Accounts
├── Pixels / Datasets
├── Apps
└── People (with role assignments per asset)
```

### Setting Up Business Assets
1. Go to business.facebook.com → Business Settings
2. Add a Page (claim ownership or create new)
3. Add an Ad Account (create new or request access)
4. Add People → assign them roles on specific assets

### Commerce Manager (Facebook/Instagram Shop)
- Create a product catalog (manual, CSV/XML feed, or eCommerce platform integration)
- Set up a Facebook Shop (products browsable on your Page)
- Enable **Checkout on Facebook/Instagram** (US only) for in-app purchases
- Connect Shopify, WooCommerce, BigCommerce, or other platforms for catalog sync
- Tag products in posts, Stories, Reels, and Live videos

## Pages & Navigation

**Page types available:**
- Regular pages (content, about, contact, etc.)
- Collection pages: Blog, Portfolio, Events, Products
- Index pages (grouped sections with shared navigation)
- Link pages (external URL in nav)
- Folder pages (dropdown navigation group)
- Cover pages (full-screen landing pages)

**Page management:**
- Pages panel (left sidebar in editor) — drag to reorder
- Duplicate pages or sections via "..." menu
- Password-protect individual pages or entire site
- Custom 301 and 302 redirects via Pages → Not Linked → URL Mappings
- Hide pages from navigation without deleting
- Custom per-page URLs (slugs)

**Navigation:**
- Main navigation, secondary navigation, footer navigation
- Drag-and-drop to reorder nav items
- Folder/dropdown menus supported
- Ability to hide header/footer per page (Advanced page settings)

## Design & Layout

**Fluid Engine editor:**
- Drag-and-drop grid-based layout with pixel-level control
- Resize, reposition, and layer any block
- Desktop and mobile views editable independently

**Design panel (site-wide):**
- Style Kit: choose fonts, color palette, imagery style, button styles, spacing
- Site-wide colors: primary, secondary, accent, background, heading
- Font pairing: heading + body font selection
- Button styles: text, shape, fill, outline, border radius

**Section-level controls:**
- Background: color, gradient, image, video
- Section dividers (shape dividers between sections)
- Padding and spacing
- Background art overlays and shapes
- Animations: section reveal, parallax effects

**Block-level controls:**
- Image effects: filters, overlays, focal point, animation
- Text highlights: hand-drawn circles, underlines, boxes
- Scale Text: maintain consistent text rendering
- Shape blocks, pinned blocks

**Mobile:**
- All sites are responsive by default
- Mobile-specific layout adjustments in Fluid Engine
- Mobile preview mode in editor

## Content Blocks

**Media:** Image, Video, Audio, Gallery (grid, slideshow, stacked)
**Text:** Text, Quote, Code
**Layout:** Spacer, Line, Shape, Button, Accordion
**Interactive:** Form, Newsletter signup, Search
**Embed:** Map, Chart, Social links, External embed
**Commerce:** Products block, Summary block
**Other:** Markdown block, Table of contents, Scrolling text

**Forms:**
- Built-in form builder with custom fields
- Submissions stored in Squarespace or forwarded via email
- Integrations: Mailchimp, Google Drive, Zapier
- API key permission: "Forms" to access via API

## Blogging

**Post workflow:** Draft → Review → Scheduled → Published
**Scheduling:** Set future publish date/time
**Organization:** Categories and tags; multiple authors
**Features:**
- Comments with threading, likes, moderation, spam filters
- Markdown Block for composing in Markdown
- RSS feed auto-generated for each blog
- Podcast support with Apple Podcasts integration
- Post excerpts and featured images

**Email integration:** Blog posts auto-reformat into Email Campaign drafts

## SEO

**Built-in SEO tools:**
- SEO title and description fields on every page and product
- URL slug customization
- Auto-generated XML sitemap (submitted to Google/Bing)
- Canonical URLs
- Structured data (JSON-LD) for products, events, and articles
- Open Graph and Twitter Card meta tags auto-populated
- Image alt text fields
- Clean semantic HTML output

**SEO checklist** (in Admin → Marketing → SEO):
- Connect Google Search Console
- Submit sitemap
- Set site title and tagline
- Configure page-level SEO descriptions

**301 redirects:** Admin → Pages → Not Linked → URL Mappings

## Domains

**Domain options:**
- Register directly in Squarespace (auto-renewing)
- Connect third-party domain (GoDaddy, Namecheap, Google, etc.)
- Transfer domain to Squarespace
- Free domain for first year on annual plans

**Domain settings:**
- DNS records management (A, CNAME, MX, TXT records)
- HTTPS/SSL: automatic, free on all sites
- WHOIS privacy: included free
- Domain forwarding
- Email forwarding aliases (free, unlimited)

**Google Workspace:** Connect custom domain email via Admin → Email → Google Workspace (paid add-on)

## Memberships & Gated Content

**Member Areas** (subscription-based gated content):
- Online Courses: video and text lessons organized in chapters
- Video Libraries: browsable paywall-protected video collections
- Member Sites: gated community pages
- Paywalled blog posts

**Setup:** Admin → Commerce → Member Areas
**Pricing:** Monthly or annual subscription per member area
**Access:** Members log in to view gated content

## Contributor Roles

| Role | Access |
|---|---|
| Administrator | Full access |
| Editor | Edit content, no billing or settings |
| Store Manager | Manage orders, inventory, coupons |
| Analytics | View analytics only |
| Billing | Billing and plan management only |

Add contributors: Admin → Settings → Permissions → Invite Contributor

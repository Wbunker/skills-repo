# TikTok LIVE

## Requirements to go LIVE

| Requirement | Value |
|-------------|-------|
| Minimum followers | **1,000** (most regions) |
| Minimum age | **18 years old** (to access LIVE Gifts/monetization) |
| Account type | Personal account (Business accounts cannot receive LIVE Gifts) |
| Account standing | Must comply with Community Guidelines; no active violations |
| Age verification | Required for monetization features |

Notes:
- The 1,000 follower minimum may vary by region
- Some creators with under 1,000 followers can unlock LIVE by contacting TikTok support or through special programs
- Going LIVE as a **guest** (not host) may have different requirements
- Government, politician, and political party accounts cannot collect Diamonds via LIVE Gifts

## Starting a LIVE

### From mobile app
1. Tap **+** (create) → swipe to **LIVE** tab
2. Add a LIVE title
3. Choose a cover image (optional)
4. Select topic/category
5. Tap **Go LIVE**

### From desktop (LIVE Studio / RTMP)
- Requires TikTok LIVE Studio desktop app or OBS/Streamlabs + RTMP stream key
- Generate stream key in: Profile → LIVE → Get stream key
- Stream key tied to session (regenerate for each LIVE)
- RTMP URL: `rtmp://push.tiktokcdn.com/live/`

## LIVE features

### Interactive tools

| Feature | Description |
|---------|-------------|
| Virtual gifts | Viewers send gifts (purchased with Coins) during LIVE |
| LIVE Q&A | Viewers submit questions; displayed as pinned prompts on screen |
| Filters and effects | Beauty filters, AR effects, background replacement |
| Polls | Real-time viewer polls during LIVE |
| Pinned comments | Host can pin key comments for visibility |
| Goal | Set a gift goal; displayed as a progress bar |
| Share screen | Show another app on screen during LIVE (desktop) |

### Multi-guest LIVE (LIVE with friends)

- Up to **20 co-hosts** simultaneously in a grid or split-screen layout
- Invites sent via LIVE multi-guest button during broadcast
- Guests can also receive gifts from viewers
- Used for interviews, collabs, watch parties, Q&A panels

### Moderators

- Host can designate trusted followers as moderators
- Moderators can mute/remove disruptive viewers, pin comments, manage Q&A

## Gifts and Diamonds

### How the gift system works

1. Viewers purchase **TikTok Coins** with real money (price varies by platform/region; approximately $1 USD = ~70 Coins)
2. During LIVE, viewers spend Coins to send **virtual gifts** (roses, hearts, lions, universes, etc.)
3. Gifts convert to **Diamonds** in the creator's account
4. Creators cash out Diamonds via PayPal or bank transfer

### Gift → Diamond → USD conversion

| Item | Value |
|------|-------|
| Base rate | 1 Diamond = $0.005 USD (0.5 cents) |
| 1,000 Diamonds (base) | $5.00 USD |
| Effective payout rate (2025) | 20%–53% of base (mission-dependent) |
| Minimum payout | ~100 Diamonds / approximately $10 USD equivalent |

### 2025 Scaled Rewards system

As of early 2025, TikTok introduced a **mission-based payout multiplier**:
- **Per-LIVE missions**: completing in-LIVE goals unlocks up to +40% payout bonus
- **Weekly missions**: completing weekly creator challenges unlocks up to +13% payout bonus
- Maximum total multiplier: **53%** (base 20% + 40% per-LIVE + 13% weekly)
- Creators who do not complete missions receive only ~20% of base diamond value

This system replaced the previous flat-rate Diamond conversion.

## LIVE analytics

Accessible in TikTok Studio → Analytics → LIVE tab.

Per-session metrics:
| Metric | Description |
|--------|-------------|
| Total viewers | Unique accounts that joined the LIVE |
| Peak concurrent viewers | Max simultaneous viewers at any point |
| Total diamonds earned | Total gift value received in the session |
| New followers from LIVE | Follows attributed to the LIVE session |
| Duration | Total LIVE session length |
| Comments | Number of viewer comments during session |
| Shares | Times viewers shared the LIVE |

Aggregate LIVE analytics available for 7/28 day windows.

## Scheduled LIVE (LIVE Events)

- Hosts can schedule a future LIVE broadcast and set a countdown
- Followers can **register** (get push notification when LIVE starts)
- Appears as an upcoming event on creator's profile
- Minimum advance scheduling: ~10 minutes
- Shows estimated audience (registered followers)

## LIVE monetization summary

| Method | How it works |
|--------|-------------|
| LIVE Gifts → Diamonds | Viewers send gifts during LIVE; creator cashes out Diamonds |
| LIVE Subscription (Super Fan) | Monthly subscribers unlock exclusive LIVE perks (see monetization.md) |
| Brand integrations | Paid LIVE sponsorships negotiated externally |
| TikTok Shop LIVE | Pin products during LIVE; viewers purchase via in-LIVE product links |

## LIVE best practices

- Go LIVE consistently at the same days/times to build habitual viewership
- Announce LIVE sessions in advance via regular videos
- Set a Gift Goal for the session to drive viewer participation
- Use Q&A to maintain interactive dialogue
- Moderate actively — remove spammers and trolls quickly
- Aim for sessions of 30–60+ minutes to maximize viewer accumulation
- LIVE session performance affects Creator Rewards eligibility indirectly (follower growth)
